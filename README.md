SkillSwap: Student Talent Exchange Platform repository information:

Repository explanation (Stars are used to show how far down folders are in the repository):

Web-App-Senior-Project- - Root folder

*Virtual-Contest-Project - Secondary root folder

**.idea - Contains files used by PyCharm, the selected Python IDE, for background usage

**documentation - Holds the plans for how the web application connects from the database to the web, the database's Entity Relationship Diagram

**Group Project Server - The main folder containing all of the web application's files that are used to maintain the server and API

***.idea - Contains files used by PyCharm, the selected Python IDE, for background usage

***.venv - The virtual environment containing all modules used within the application

***__pycache__ - Used by PyCharm to hold already compiled Python files for optimization

***routers - Contains Python files for initialization, endpoints, and page renderings

***static - Contains all images, CSS files, and JavaScript files

***templates - Contains HTML page templates that Jinja2Templates uses to reduce redundancy

***database.py - Connects the Python API to the database

***main.py - The main Python file, Defining routers, templating, and the application itself

***models.py - Contains Python classes that become tables within the database

***requirements.txt - Lists all required modules to run the application
