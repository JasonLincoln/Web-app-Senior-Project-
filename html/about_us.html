SkillSwap: Student Talent Exchange Platform is a project developed by four programmers wanting to make an app for tutors and those in need of tutoring. SkillSwap provides that service through our messaging system and the ability to schedule tutoring/studying sessions with other users. We strived for easy user access with the use of a 'swipe-left-swipe-right' tutor search for those not interested in long tables of information. However, those proficient in querying a database such as ours will find they can still surf through our search interface for the perfect tutor rather than use the generally accepted mobile format we have given access to. Skillswap also has the ability to work on whatever device you wish, whether that is a personal computer, laptop, phone, or tablet, it can be ran through the web on any of them. But, that's enough about the website, there is still more to learn about us, the developers. The developers of SkillSwap are all highschool students who started this website for a school project that also served as a submittable piece in a web application concert. All of them had their own strengths, and were given tasks based off of them. Half worked on the front end of the site that is currently being browsed. Meanwhile, one of them acted as the core back-end designer, giving users the ability to sign up, log in, schedule sessions, and message one another, with the use of a database that stores the information that allows for users to perform the former tasks. The last member took to preparing what would be shown in the contest this website was submitted for, while also filling in where needed through the development process.
